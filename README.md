# launchd-plists
A collection of launchd plist files on macOS.

## Setup
```shell
./setup
```

## Other information
[LaunchControl](https://www.soma-zone.com/LaunchControl/) is useful to launch application via GUI.

```shell
brew install --cask launchcontrol
```
